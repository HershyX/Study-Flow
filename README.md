StudyFlow – Student Productivity Dashboard

StudyFlow is a lightweight web-based productivity dashboard that helps students manage their study time, track progress, and stay organized. It includes features like subject tracking, study session timers, daily goals, a Pomodoro timer, task management, and study history.

The application is built using HTML, CSS, and JavaScript, and stores all data locally in the browser using LocalStorage, so it runs without any backend or database.

Features

Study progress dashboard

Daily study goal tracking

Subject management with weekly targets

Real-time study session timer

Pomodoro focus timer

Task management with priority levels

Study session history

Study streak tracking

Technologies Used

HTML

CSS

JavaScript

LocalStorage

How to Run

Download the project files.

Keep index.html, styles.css, and script.js in the same folder.

Open index.html in any browser.
